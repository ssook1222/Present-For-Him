### 📍 프로젝트 목적

1주년 기념 남자친구를 위한 웹 서비스 개발 프로젝트

### 📍 개발 기간

24.03 ~ 24.04

### 📍 서비스 개요

텍스트 어드벤처 형태의 선택지 형 웹 서비스 게임으로, 모든 텍스트들이 남자친구와 저 사이에 있었던 일을 기반으로 구성되었습니다.         
엔딩은 총 4개로, 진엔딩 1개 / 노말엔딩 1개 / 배드엔딩 2개로 구성되어 있습니다.          

`기술 스택`       
- React.js
- Vercel.app
